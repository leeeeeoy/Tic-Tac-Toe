# tic_tac_toe

Flutter로 만든 Tic-Tac-Toe 게임입니다. 최대 6x6 배열까지 진행 가능합니다.

Flutter version: 3.19.2

### 화면구성

#### HomeScreen
- 처음 시작할 때 화면입니다.

#### SettingScreen
- 게임 옵션을 선택하는 화면입니다.

#### GameScreen
- 실제 게임을 진행하는 화면입니다.

#### RecordsScreen
- 저장된 게임 목록을 보는 화면입니다.

#### RecordDetailScreen
- 저장된 게임의 상세 정보를 보는 화면입니다.
